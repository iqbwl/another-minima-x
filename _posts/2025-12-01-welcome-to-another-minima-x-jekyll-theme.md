# Welcome to Another Minima X Jekyll Theme!

This is your first post using the **Another Minima X Jekyll Theme**. This theme is designed to provide a clean, modern look for your personal blog or digital notebook.

## Features Showcase

Let's explore some of the features this theme offers:

### Beautiful Typography

The theme uses the **Inter** font family, which provides excellent readability across all devices. Text is carefully sized and spaced for comfortable reading.

### Dropdown Menus

The theme supports nested navigation menus, allowing you to organize your links hierarchically in the header.

### Code Highlighting

The theme includes syntax highlighting for code blocks:

```python
def hello_world():
    print("Hello, World!")
    return True

if __name__ == "__main__":
    hello_world()
```

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
  return true;
}

greet('World');
```

## Getting Started

To create your own posts:

1. Create a new file in the `_posts` directory
2. Name it with the format: `YYYY-MM-DD-title.md`
3. Add frontmatter with title, slug, author, and categories
4. Write your content in Markdown
5. Save and build your site!

## Customization

Feel free to customize this theme to match your style. You can:

- Modify colors in the SCSS variables
- Override layouts and includes
- Add your own custom CSS
- Configure the navigation menu
