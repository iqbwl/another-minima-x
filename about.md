# About This Theme

Welcome to the **Another Minima X Jekyll Theme** demo site! This theme is designed for personal blogs and digital notebooks, featuring a clean and modern design.

## Features

This theme includes:

- **Glassmorphism Header**: A modern, frosted-glass effect header that adapts to your content
- **GitHub-Style Markdown**: Beautiful markdown rendering that matches GitHub's style
- **Clean Typography**: Uses the Inter font for excellent readability
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Disqus Integration**: Built-in support for comments
- **SEO Optimized**: Includes Jekyll SEO Tag for better search engine visibility

## Getting Started

To use this theme for your own site:

1. Install the theme (see [README](https://github.com/iqbwl/another-minima-x))
2. Customize `_config.yml` with your information
3. Start writing posts in the `_posts` directory
4. Deploy to your favorite hosting platform

## Customization

You can customize the theme by:

- Modifying colors and styles in your own SCSS files
- Overriding layouts and includes
- Adding your own custom CSS
- Configuring navigation menu items

## Support

If you encounter any issues or have questions, please open an issue on [GitHub](https://github.com/iqbwl/another-minima-x/issues).
